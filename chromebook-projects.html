1. Installing LibreWolf on Chromebook

Steps:

  1. Enable Linux (Crostini) on your Chromebook
  2. Open Terminal and run: sudo apt install flatpak -y
  3. Add Flathub repository: sudo flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
  4. Install LibreWolf: sudo flatpak install flathub io.gitlab.librewolf-community -y
  5. Launch LibreWolf from your Linux apps menu

2. Setting Up Steam on Chromebook (No Root)

Steps:

  1. Enable Linux (Crostini) on your Chromebook
  2. Open Terminal and run: sudo apt update && sudo apt upgrade -y
  3. Run: sudo dpkg --add-architecture i386
  4. Run: sudo apt update
  5. Run: sudo apt upgrade
  6. Run: sudo apt-get install --reinstall libgl1-mesa-glx:i386
  7. Download Steam .deb file from the official website
  8. Install Steam using: sudo dpkg -i steam_latest.deb
  9. Launch Steam from your Linux apps menu

4. Setting Up Linux Desktop on Chromebook (No Root)

Steps:

  1. Enable Linux (Crostini) on your Chromebook
  2. Open Terminal and run: sudo apt update && sudo apt upgrade -y
  3. Install Xfce desktop: sudo apt install task-xfce-desktop -y
  4. Install Xephyr: sudo apt install xserver-xephyr -y
  5. Create a script to launch the desktop: sudo nano /usr/bin/gox
  6. Add the following content to the script: 
    Xephyr -br -fullscreen -resizeable :40 &
sleep 5
sudo -u $USER DISPLAY=:40 startxfce4 &
  7. Save the script and make it executable: sudo chmod +x /usr/bin/gox
  8. Run the desktop environment by typing 'gox' in the terminal

5. Installing Kali Linux on Chromebook (No Root)

Steps:

  1. Enable Linux (Crostini) on your Chromebook
  2. Open Terminal and run: sudo apt install nano -y
  3. Edit sources list: sudo nano /etc/apt/sources.list
  4. Add this line: deb http://http.kali.org/kali kali-rolling main contrib non-free
  5. Download Kali key: sudo wget https://archive.kali.org/archive-key.asc -O /etc/apt/trusted.gpg.d/kali-archive-keyring.asc
  6. Update system: sudo apt update -y && sudo apt full-upgrade -y
  7. Install Kali: sudo apt install kali-linux-default -y
  8. Set up Xephyr for GUI (follow steps from Linux Desktop setup)
